# SIRSimulation

SIRSimulation is a small R package for running the existing SIR model
simulation functions.

Load the package with:

```r
library(SIRSimulation)
```

Public functions:

- `create_matrix()`
- `create_crnr_matrix()`
- `create_cntr_matrix()`
- `simulate_sir()`
- `simulate_inf_seq()`
- `simulate_many_runs()`
- `multiple_run_heatmap()`
